Welcome to the GVL Dashboard! The GVL Dashboard is a portal through which you can access all services on the GVL workbench.

Available Services

Service Name Description Status Access Link
1 Galaxy Galaxy is an open, web-based platform for accessible, reproducible, and transparent computational biomedical research.

http://localhost/galaxy
2 Cloudman CloudMan is a cloud manager that orchestrates the steps required to provision and manage compute clusters on a cloud infrastructure. Use Cloudman to start and manage your Galaxy service, and to add additional nodes to your compute cluster.

http://localhost/cloud
3 Lubuntu Desktop This is a lightweight desktop environment through which you can run desktop applications on your virtual machine. You can also access the GVL command line utilities through the desktop.

http://localhost/vnc
4 GVL GenomeSpace Genomespace is like a dropbox for GenomeData. Use Genomespace to store all your Genomics Data securely and redundantly.

https://genomespace.genome.edu.au
5 iPython Notebook The IPython Notebook is a web-based interactive computational environment where you can combine code execution, text, mathematics, plots and rich media into a single document.

http://localhost/ipython_notebook
6 R-Studio R Studio IDE is a powerful and productive user interface for R

http://localhost/rstudio
7 SSH You can login to your virtual machine remotely through an SSH client.

ssh -i <your_cloudman_keypair> ubuntu@localhost
8 GVL UCSC Genome Browser The UCSC Genome Browser allows you to visualise genome sequence data from a variety of vertebrate and invertebrate species and major model organisms, integrated with a large collection of aligned annotations.

https://ucsc.genome.edu.au/
9 GVL Tutorials Access GVL tutorials and other learning resources.

https://genome.edu.au/wiki/Learn
